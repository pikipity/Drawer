小画板
======

这是我的一个 processing 的练习作，旨在练习 Processing 的基本命令和熟悉 Java 的语法。processing 可以在[这里](https://processing.org/download/)下载。

左侧选择工具，右侧选择画笔粗细和颜色。按"e"键结束当前工具输入（画直线、方框、圆形和不规则图形时有效）；按"r"键画布清零。
